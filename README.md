# ELITE-E-Commerce

## Description
This is an e-commerce website where users can browse products and place orders.  
It also includes an admin system to manage the platform.

## Features
- User login and signup
- Add to cart
- View products
- Order system
- Admin dashboard
- Super Admin control

## Admin System
- Super Admin can manage the entire website
- Super Admin can approve or allow new admins
- Normal Admin can log in and sell their products

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Run: python app.py
3. Open: http://127.0.0.1:5000
